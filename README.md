# Autonomous AI Research Agent

A LangGraph-powered AI research assistant that performs live web research and generates structured reports for competitor analysis, market research, and trend analysis.

## Features

- Live web search with Tavily
- LangGraph-based multi-step workflow
- OpenAI-powered planning and report generation
- Research plan generation
- Structured markdown reports
- Source-backed findings
- Streamlit UI
- Markdown export

## Tech Stack

- Python
- Streamlit
- LangChain
- LangGraph
- Tavily API
- Groq API

## Project Workflow

1. User enters a research topic
2. Agent creates a research plan
3. Agent runs live web searches
4. Findings are summarized
5. Final report is generated in markdown
